🚀 QUICK START (Follow These Steps):
════════════════════════════════════════════════════════════════════════════════

Step 1: Install Dependencies
   pip install -r requirements.txt

Step 2: Run ML Pipeline (20-30 minutes)
   python backend/src/data_transform.py       # 2-5 min
   python backend/src/training.py             # 10-15 min
   python backend/src/evaluation.py           # 3-5 min

Step 3: View Results
   mlflow ui                                  # Opens http://localhost:5000

Step 4: Start Backend (Terminal 1)
   python -m uvicorn backend.src.app:app --reload
   # Runs on http://localhost:8000

Step 5: Start Frontend (Terminal 2)
   streamlit run frontend/app.py
   # Opens http://localhost:8501

Step 6: Analyze Messages!
   • Go to http://localhost:8501
   • Paste an email or message
   • Click "Analyze"
   • See risk score + red flags

🎯 KEY FEATURES:
════════════════════════════════════════════════════════════════════════════════

Red Flag Detection (6 Categories):
   🚨 Urgency           Urgent language → urgent, immediate, act now
   🔗 Suspicious Links  Call-to-action → click, verify, confirm
   💰 Money Requests    Payment request → payment, wire, fee
   🔐 Personal Info     Sensitive data → password, SSN, PIN
   🎭 Deception         Spoofing      → fake, spoofed, impersonate
   📧 Generic Greeting  Not personalized → dear customer, dear user

Risk Scoring (0-100%):
   🟢 0-33%             Safe - Legitimate message
   🟠 33-66%            Suspicious - Needs caution
   🔴 66-100%           Danger - Likely phishing

Red Flag Confidence:
   Each flag has 0-100% confidence based on:
   • Number of matching keywords
   • Model's prediction confidence
   • Combined phishing indicators

🏗️ ARCHITECTURE:
════════════════════════════════════════════════════════════════════════════════

Data Flow (Training):
   Raw CSV
     ↓
   [data_transform.py] Clean text
     ↓
   Processed CSV
     ↓
   [training.py] Train 15 models with CV
     ↓
   MLflow (Logged models)
     ↓
   [evaluation.py] Comprehensive metrics
     ↓
   evaluation_report.csv

Data Flow (Inference):
   User Message (Streamlit)
     ↓
   HTTP POST /analyze
     ↓
   [FastAPI Backend]
     ↓
   Load model from MLflow
     ↓
   Clean text → Vectorize → Predict
     ↓
   Detect red flags
     ↓
   Calculate risk score
     ↓
   Return JSON response
     ↓
   Display in Streamlit
     ↓
   Risk gauge + Red flags + Explanation
